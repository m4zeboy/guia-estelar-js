# Tipos de dados

Conforme o ECMASCRIPT temos 9 tipos de dados:

* Data Types 
    * Primitives / Primitive value;
    * Structural;
    * Structural Primitive.

## Primitivos

* String;
* Number;
* Boolean;
* undefined;
* Symnol;
* BigInt;

## Estruturais

* Object
    * Array;
    * Map;
    * Set;
    * Date;
    * ...

## Primitivo estrutural / Structural root primitive

* null
